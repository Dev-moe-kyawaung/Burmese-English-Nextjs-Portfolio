# android-senior-developer-portfolio

Professional portfolio website for an Android Senior Developer, built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Burmese/English language support.

## Features

- Fixed top navigation.
- Animated hero section.
- Advanced about section with timeline.
- Certificates collection.
- Mail collection.
- Lovable WPA links collection.
- GitHub collections.
- Organizations section.
- My create app collection.
- Smooth scroll and UX enhancement.
- Performance optimization.
- Burmese language support.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis
- i18next / react-i18next
